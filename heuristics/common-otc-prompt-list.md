# Common OTC Prompt List

## Description

A standard set of commonly used over-the-counter medications that experts routinely prompt patients about, because patients often do not consider these to be "medications."

## Prompt Categories

| Category | Examples |
|---|---|
| **Analgesics** | Paracetamol, ibuprofen, aspirin, codeine-containing products |
| **Gastrointestinal** | Antacids, laxatives, anti-diarrhoeal agents |
| **Sleep aids** | Melatonin, antihistamines (e.g., doxylamine), herbal sleep products |
| **Cold and flu** | Decongestants, cough suppressants, combination products |
| **Topical products** | Corticosteroid creams, antifungal creams, eye drops |
| **Supplements** | Vitamins, minerals, fish oil, probiotics |

## Application

Used during **[PRC-001](../procedures/collect-medication-information.md)** Step 2, Question 6 (prompted recall): *"To make sure that I have not missed anything, do you take any of the following...?"*

## Rationale

Patients frequently omit OTC medications from their self-report because they do not consider them "real medications." These products can cause significant drug interactions and adverse effects.
