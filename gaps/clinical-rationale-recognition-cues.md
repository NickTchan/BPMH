# Expert Cues for Recognising Clinically Obvious Medication Changes

## Gap Description

Experts can rapidly triage medication discrepancies into "obviously intentional" vs. "needs investigation" based on the patient's clinical context. The **specific clinical cues** that trigger this recognition are not documented.

## Why This Matters

This pattern recognition is what separates experts from novices. Novices treat all discrepancies equally, requiring prescriber confirmation for each one. Experts save time and focus attention where it matters by instantly recognising clinically obvious changes.

## What We Know

- Clinical context (diagnosis, investigation results) informs discrepancy classification
- Some changes are "clinically obvious" (e.g., ceasing metformin in AKI)

## What We Don't Know

- The full set of clinical cue → medication change patterns that experts recognise
- How experts handle borderline cases where the clinical rationale is plausible but not certain
- Whether this pattern recognition is medication-specific or diagnosis-specific

## CTA Priority: 4

This is a core perceptual discrimination that novices universally miss and that directly affects efficiency and safety.
